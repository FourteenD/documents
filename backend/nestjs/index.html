# 介绍

如果之前没有完整的后端经验，那么强烈建议耐心把文档过一遍。

# CLI

Nest CLI 可以更好的帮助管理项目，这是一开始就需要知道的事情。

npm i @nestjs/cli -g

# 工作流程

一切从查看帮助开始：

nest -h

创建项目：

nest new user-crud

三连：

# 模块
nest g mo user
# 控制器
nest g co user
# 服务
nest g s user

比三连更强的 res：

# 附加 dto 及 entities
nest g res user
